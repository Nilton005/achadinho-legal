Política de Privacidade

Última atualização: 08 de maio de 2025

Esta Política de Privacidade descreve como o Achadinhopolegazinha coleta, usa e protege as informações dos usuários ao utilizar nosso aplicativo de análise de tendências do TikTok.

1. Informações que Coletamos

Podemos coletar os seguintes tipos de informação:

  • Dados de autenticação: tokens de acesso fornecidos pelo TikTok para identificar o usuário autenticado
  • Dados de uso: como você interage com as funcionalidades do aplicativo
  • Dados públicos do TikTok: informações de vídeos, produtos e métricas públicas acessadas via API oficial
  • Preferências do usuário: configurações salvas localmente no seu dispositivo

2. Como Usamos as Informações

As informações coletadas são utilizadas exclusivamente para:

  • Autenticar o acesso às APIs do TikTok
  • Exibir análises e tendências personalizadas
  • Melhorar a experiência de uso do aplicativo
  • Salvar preferências e configurações do usuário

3. Armazenamento de Dados

As suas credenciais e configurações são armazenadas localmente no seu dispositivo (localStorage do navegador) e não são transmitidas para servidores externos de nossa propriedade. Os tokens de acesso do TikTok são utilizados apenas para comunicação direta com as APIs oficiais do TikTok.

4. Compartilhamento de Dados

Não vendemos, alugamos nem compartilhamos suas informações pessoais com terceiros, exceto:

  • Quando necessário para comunicação com as APIs oficiais do TikTok
  • Quando exigido por lei ou ordem judicial
  • Para proteger os direitos e segurança dos usuários

5. API do TikTok

Este aplicativo utiliza a API oficial do TikTok for Developers. Ao usar o Achadinhopolegazinha, você também está sujeito à Política de Privacidade do TikTok. Os dados acessados via API são dados públicos disponibilizados pela plataforma.

6. Cookies e Armazenamento Local

Utilizamos o armazenamento local do navegador (localStorage) para salvar suas preferências e tokens de autenticação. Nenhum cookie de rastreamento de terceiros é utilizado.

7. Segurança

Adotamos medidas técnicas para proteger as informações dos usuários. No entanto, nenhum sistema é 100% seguro. Recomendamos que você mantenha seus tokens de acesso confidenciais e não os compartilhe com terceiros.

8. Seus Direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), você tem direito a:

  • Acessar os dados que possuímos sobre você
  • Solicitar a correção de dados incorretos
  • Solicitar a exclusão dos seus dados
  • Revogar o consentimento a qualquer momento

9. Menores de Idade

Este aplicativo não é destinado a menores de 18 anos. Não coletamos intencionalmente informações de crianças ou adolescentes.

10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas através do próprio aplicativo. O uso continuado após as alterações implica aceitação da nova política.

11. Contato

Para exercer seus direitos ou tirar dúvidas sobre esta política, entre em contato pelo e-mail disponível no aplicativo.
